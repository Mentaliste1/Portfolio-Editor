Héberger mon Portfolio
Guide complet — Gratuit en 5 minutes
Pourquoi GitHub Pages ?
GitHub Pages est 100% gratuit, fiable, et donne une URL professionnelle du type tonnom.github.io/portfolio. Parfait pour partager avec des recruteurs ou des clients.
Vidéo tutoriel (2 min)
Tutoriel : Héberger un fichier HTML sur GitHub Pages gratuitement
Étapes détaillées
1
Télécharger ton portfolio
Clique sur "Publier le Portfolio" dans ce même menu Exporter. Un fichier Mon_Portfolio_Publie.html sera téléchargé sur ton appareil.
2
Créer un compte GitHub gratuit
Va sur github.com et crée un compte gratuit avec ton adresse email. Si tu en as déjà un, passe directement à l'étape suivante.
Créer un compte GitHub
3
Créer un nouveau repository
Une fois connecté sur GitHub, clique sur le bouton vert "New" ou "+" en haut à droite → "New repository".

• Nom du repository : portfolio
• Visibilité : Public (obligatoire pour GitHub Pages gratuit)
• Coche : "Add a README file"
• Clique "Create repository"
4
Uploader ton fichier portfolio
Dans ton repository GitHub :

• Clique "Add file" → "Upload files"
• Glisse-dépose ton fichier Mon_Portfolio_Publie.html
• ⚠️ Renomme-le en index.html avant d'uploader
• Clique "Commit changes"
5
Activer GitHub Pages
• Dans ton repository, clique sur "Settings" (onglet en haut)
• Dans le menu gauche, clique "Pages"
• Sous "Source", sélectionne Deploy from a branch
• Branch : main → / (root)
• Clique "Save"
✓
Ton portfolio est en ligne ! 🎉
Après 1 à 2 minutes, ton portfolio sera accessible à l'URL :
https://ton-username.github.io/portfolio

Tu peux partager ce lien sur ton CV, LinkedIn, ou l'envoyer directement à des recruteurs.
